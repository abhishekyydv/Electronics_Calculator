How This Calculator Works

This calculator uses the following formulas based on your input parameters:

  • Duty Cycle (D): D = Vout / (Vin × Efficiency)
    where Efficiency is in decimal (e.g., 0.8 for 80%)
  • Input Power (Pin): Pin = (Vout × Iout) + (Vout × Iout × (1 − Efficiency))
  • Input Ceramic Capacitance (Cin): Cin = (Iout × D × (1 − D)) / (fsw × Vripple)
  • Ripple Voltage RMS (VRMS): VRMS = Vripple × 0.22867
  • Ripple Current (Iripple): Iripple = Iout × 0.3
  • Capacitor Current (Icap): Icap = (VRMS / ESR) × 1000
  • Capacitor Power Dissipation (Pdiss): Pdiss = (Icap)2 × ESR / 1000
  • Input Transient Current (Itrans): Itrans = 0.75 × Iout
  • Input Bulk Capacitor (Cin,bulk): Cin,bulk = (1.21 × Itrans2 × L) / (Vripple2 × 1000)
  • Output Impedance (Zout): Zout = (Vripple / Itransient) × 1000 (in mΩ)
  • Output Bulk Capacitor (Cout,bulk): Cout,bulk = (1.21 × Itransient2 × L) / (Vripple2 × 1000)

Variables used:

  • Vin: Minimum input voltage (V)
  • Vout: Output voltage (V)
  • Iout: Output current (A)
  • Efficiency: Efficiency in decimal (e.g., 0.8 for 80%)
  • fsw: Switching frequency (Hz)
  • Vripple: Ripple voltage peak-to-peak (V)
  • ESR: Equivalent Series Resistance (mΩ)
  • L: Inductor value (nH)
  • Itransient: Output transient current (A)
